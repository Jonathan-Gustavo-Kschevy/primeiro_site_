Hospedagem GitHub

Para criar um site no GitHub.com devemos ter uma conta válida e um repositório com um arquivo index.html

Hospedendo um Site

  1. Criar um repositório;
  2. Publica o repositório utilizando o Publish;
  3. Abrir o repositório no VSCode;
  4. Criar o index.html;
  5. Criar a varsão usando commit;
  6. Enviar para o repositório remoto usando push;
  7. Entrar no GitHub.com, acessar o repositório -> Settings -> Pages -> Escolher a branch do source
  8. Clicar na URL gerada.

Branches

São ramificações do seu projeto. O projeto principal, em geral, fica na Branch main. A Branch main deve ter a versão final e funcional do seu projeto. O projeto pode ter branches secundárias. Clique aqui para saber mais sobre branches secundárias
